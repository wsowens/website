Projects

These are some of the more notable projects I have completed as an undergraduate.

Muddy Swamp

MuddySwamp is a collaborative UF-themed multi-user dungeon. (Think World of Warcraft but text-based.

I launched this project shortly after I was elected President of the Open Source Club at UF.

Antaeus

Antaeus is a Mandelbrot set renderer that I made in a weekend to learn Simple DirectMedia Layer.


Because this project is entirely CPU-rendered, it's quite slow when compared to hardware-accelerated renderers.



One redeeming trait: thanks to SDL, this project runs everywhere.
